# Recipe App

A simple recipe app built with JavaScript using localStorage for persistent data

![Preview](./preview.png)

## Features
- Add and remove recipes
- Filter by category and difficulty
- Search by name or ingredients
- Persistent data using localStorage

## Tech
- HTML
- CSS
- JavaScript

## About the project
This project was built as part of a course assignment. The focus was on working with DOM manipulation, event handling and storing data in localStorage

## Implementation
- Recipes are stored as objects in an array and saved in localStorage
- Dynamic DOM rendering is used to create recipe cards
- Filtering logic is handled through a custom filter function
- A class-based structure is used to represent each recipe

## Notes
- Test recipes are included and only added if localStorage is empty
- Current version is Swedish only
